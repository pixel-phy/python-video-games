📚 Temario completo de Python (16 semanas)

Semana 1	Variables, tipos de datos y operadores	int, float, str, bool, operadores aritméticos, input(), print(), f-strings, type(), conversión de tipos (int(), str()), strip(), manejo básico de errores con try/except, raise
Semana 2	Condicionales	if, elif, else, comparaciones (==, !=, <, >, <=, >=), operadores lógicos (and, or, not), anidamiento, operador ternario (x if cond else y)
Semana 3	Bucles	while, for, break, continue, else en bucles, bucles anidados, contadores y acumuladores, range()
Semana 4	Strings (cadenas)	Indexación, slicing, métodos (upper, lower, strip, split, join, find, replace, startswith, endswith), formato con format() y f-strings
Semana 5	Listas	Creación, indexación, slicing, métodos (append, remove, pop, insert, sort, reverse, index, count), recorridos, list comprehension, copia vs referencia
Semana 6	Tuplas	Inmutabilidad, desempaquetado, métodos (count, index), comparación con listas, tuplas con un solo elemento
Semana 7	Conjuntos (sets)	Creación, elementos únicos, métodos (add, remove, discard, clear, union, intersection, difference), operadores (|, &, -, ^), frozenset
Semana 8	Diccionarios	Claves-valor, métodos (keys, values, items, get, pop, update, setdefault), recorridos, diccionarios anidados, dict comprehension
Semana 9	Funciones (parte 1)	Definición, parámetros (posicionales, nominales, por defecto), retorno, return vs print, scope local/global, global, documentación (docstrings)
Semana 10	Funciones (parte 2)	*args, **kwargs, funciones lambda, funciones anidadas, funciones como objetos, recursión básica
Semana 11	Manejo de errores y excepciones	try, except, else, finally, tipos de excepciones (ValueError, TypeError, IndexError, KeyError, ZeroDivisionError), raise, excepciones personalizadas
Semana 12	Archivos (files)	open(), lectura (read, readline, readlines), escritura (write, writelines), modos (r, w, a, r+), with (context managers), manejo de rutas
Semana 13 - 14 y 15	Clases y POO (básico)	class, __init__, self, atributos de instancia/clase, métodos de instancia/clase/estáticos, encapsulación (convención _), __str__, __repr__, herencia básica
Semana 16	Proyecto integrador	Combinar TODOS los temas anteriores en un programa completo
